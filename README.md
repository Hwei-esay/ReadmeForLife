# ReadmeForLife

>## 起因
>在做一些小项目的时候发现自己常常会忘记一些有用的命令，自己会把这些命令写到readme文件中，这样方便随时查看。那么我们对于人生这个大项目而言，都会有一些有用的经验、方法等，我们同样也可以将这些方法汇总到一个readme文件中，甚至可以将许多不同的人的经验汇总到一起，形成一个迭代的参考文档，这样将会提供一个有价值的参考，并且价值将会随迭代越滚越大。

一个好的Readme文件能够帮助人们快速了解项目并及时得到必要的帮助。对于拥有各种各样的人生的我们而言，虽然谁都不知道我们为什么要踏上这段旅程，也不知道我们的终点将位于哪里，但是拥有基于不同视角的思考和经验，基于这些经历、经验、道理，我们可以将我们的经验汇总起来，以引导尚处于幼年和青年时期的人，至少提供一些有依据的参考，这样才能进行有效的迭代。通过有效的迭代可以大大减少我们的迷茫，为我们指明一条或许多条较为明晰的方向。随着互联网的发展，人们可以将自己的人生经验发布到广阔的互联网，让同辈及后辈能够少走一些弯路。
